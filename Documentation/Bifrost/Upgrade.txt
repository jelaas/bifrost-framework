 Upgrading existing installation
---------------------------------

Unpacking the distribution tarball:
$ remount w
$ mount LABEL=bifrost /mnt
$ cd /mnt
$ tar xf /bifrost-M.N-N.tar.gz
$ cd /
$ umount /mnt
$ bifrost-upgrade
$ remount r
$ sync
$ reboot

If your previous Bifrost version was below 7.1 you may wish to install the grub2 boot loader.
See Install.txt for instructions.
